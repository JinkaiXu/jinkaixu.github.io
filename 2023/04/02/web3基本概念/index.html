Web3 基本知识

Web3 基础概念：

基础概念 概念解析
Web3 一个基于区块链的互联网新形态的构想，包含了去中心化和基于代币的经济学的概念。
Blockchain 区块链，一种去中心化、无法篡改的信息记录系统，是加密货币、NFT 等概念的底层技术。
Bitcoin 比特币，世界上第一个被广泛应用的加密货币。
Ethereum 以太坊，是一种区块链，支撑着 dAPP 的发展；衍生出加密货币 Ether（以太）。
Crypto 又称 Cryptocurrency，加密货币，一种去中心化的数字货币。
dAPP Decentralized App，指运行在区块链而非服务器上的应用。
NFT Non-Fungible Token，一种在链上被记录的、有所有权的数字资产。
DAO Decentralized Autonomous Organization，去中心化的自治组织。
Metaverse 元宇宙，一种由科技手段创造的虚拟世界的构想。
DeFi Decentralized Finance，去中心化金融系统。
Token 代币，可以理解为加密货币、NFT 等数字资产的统称。
GameFi Game + DeFi，中文说法链游，游戏中的金融系统可通过加密货币和 NFT 映射到现实。

常见平台介绍：

平台 平台介绍
Discord 原为游戏社群工具，后出圈成为 Web3 社群普遍使用的交流工具。
Telegram 另一个 Web3 社群常用的交流工具。
OpenSea 常用的 NFT 交易平台。
Metamask 常用的数字货币钱包。
Coinbase 常用的数字货币交易所。
a16z 硅谷知名 VC，近年来因投了很多 Crypto / Web3 项目，积累了较大影响力。
CryptoPunks NFT 代表项目之一，特点是像素画风。
BAYC NFT 代表项目之一，全称 Bored Ape Yacht Club，特点是猿猴。
Axie Infinity GameFi 代表之一，衍生出虚拟货币 AXS。
CryptoKitties GameFi 代表之一，也是最早尝试将区块链技术应用于娱乐休闲的尝试之一。

用户行为：

行为 行为定义
挖矿 指生成加密货币和验证新交易的过程
私钥 保护数字资产的密码
Mint（铸造） 指数字艺术品上链成为 NFT 的过程
白名单 / WL 指能提前 / 以折扣价购买 NFT 的机会，一般被 NFT 项目方用来运营社群
空投 指加密货币 / 代币 / NFT 等的免费发放
Roadmap 路线图，项目方用于描述项目计划。跟常规意义相同
Gas fee 交易手续费，支付给矿工（帮你在链上完成交易的人）
